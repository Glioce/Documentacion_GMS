directory_create
Crea un directorio con el nombre dado.

Sintaxis

directory_create(dname);

Argumentos

ArgumentoDescripción
dnameEl nombre del directorio a crear.

Descripción

Esta función crea un directorio con el nombre dado dentro del área local de guardado (sandbox).

Devuelve

Nada

Ejemplo

directory_create("Levels/Downloaded_Games");
Se crea un directorio llamado Downloaded_Games en el directorio Levels dentro del área local de guardado (sandbox).
