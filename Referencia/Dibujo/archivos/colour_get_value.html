colour_get_value
Obtiene el valor HSV de un color dado

Sintaxis

colour_get_value(col);

Argumentos

Argumento Descripción
col El color al cual obtener el valor.

Descripción

Esta función devuelve el valor de un color, el cual representa la "luminosidad" de dicho color, siendo que 0 significa "sin nada de luz" (haciéndose color negro), y 255 significa "totalmente iluminado".
