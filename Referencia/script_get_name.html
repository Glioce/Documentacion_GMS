script_get_name
Devuelve el nombre del script con el índice dado.

Sintaxis

script_get_name(scr);

Argumentos

ArgumentoDescripción
scrEl índice del script.

Descripción

Esta función devuelve el nombre, como cadena de texto, del script especificado. Este nombre es el que ha sido establecido para el script en el árbol de recursos del IDE Game Maker Studio.

Devuelve

Cadena de texto.

Ejemplo

nombre = script_get_name(scr_miScript);
Si el script scr_miScript existe con el mismo nombre, la función devolverá "scr_miScript".
