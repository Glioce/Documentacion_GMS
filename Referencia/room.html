room
El índice de la room actual.

Sintaxis

room;

Descripción

Esta variable contiene el índice de la room actual en la que el juego se encuentra. Esto no es una variable de sólo lectura, sin embargo cambiando esto no cambiará el valor del índice para la room actual, sino que cambiará la room para que coincida con el índice al que ha establecido la variable. Debe tener cuidado al hacer esto, ya que si el índice al que se cambia la variable no es válido el juego lanzará un error y se cerrará. En general, es mucho mejor usar room_goto() para cambiar de room.

Devuelve

Número Real

Ejemplo

if room_next(room) != -1
    {
    room_goto(room_next(room));
    }
El código anterior verificará si la room siguiente existe y si es así, irá a ella.
