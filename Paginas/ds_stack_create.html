ds_stack_create
Crea una nueva pila, regresando su id.

Síntaxis

ds_stack_create();

Argumentos

Ninguno

Descripción

Esta función permite crear una nueva estructura de datos de tipo Pila y regresa el valor de su id. Este valor debe ser almacenado en una variable para poder usar la pila en operaciones futuras.

Devuelve

Número real

Ejemplo

stack = ds_stack_create();
El código anterior crea una nueva estructura de tipo Pila y establece su índice en la variable "stack".
